Comment exécuter l'application:
Prérequis:
- Python insatallé
------------------------------------------------------------------------------------------------------
Windows:
1) Décompresser l'archive zip
2) Placer la base de donné "inginious.sqlite" (elle doit avoir ce nom) dans le dossier GroupeB8
3) Ouvir l'invite de comande (cmd)
4) Taper "cd <Le chemin du fichier GroupB8>"
   Il devrait maintenant être affiché quelque chose comme ..\Project\GroupeB8> dans votre terminal
5) Taper "py -m venv venv" pour créer un environement virtuelle
6) Taper "venv\Script\activate" pour activer l'environement virtuelle
   "(venv)" doit a présent apparaitre devant la ligne de commande.
7) Taper "pip install flask" pour installer flask
8) Taper "FLASK_APP=flaskr"
9) Taper "FLASK_ENV=development"
10) Taper "flask run" pour lancer l'application
11) Taper "127.0.0.1:5000" dans votre navigateur
Vous devriez voir apparaître la page acceuil de notre site
------------------------------------------------------------------------------------------------------
Pour tout OS faites les 2 premières étapes pour l'insatalation Windows puis consultez:
 https://flask.palletsprojects.com/en/1.1.x/installation/#installation
